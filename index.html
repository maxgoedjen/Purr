Purr

Growl style that forwads everything to Notification Center.

Download this project as a .zip file Download this project as a tar.gz file

Notification Center is Awesome.

But a lot of apps still don't support it. A of apps do, however, support Growl. Purr helps bridge the gap. Purr will take any notifications sent to Growl, and forward it straight to Notification Center.
